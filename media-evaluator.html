📋 Configuration Google Drive (5 min)
  1. Allez sur console.cloud.google.com
  2. Créez un projet (ou sélectionnez-en un existant)
  3. Activez l'API Google Drive dans "APIs & Services"
  4. Créez des identifiants → OAuth 2.0 Client ID → Type : Application Web
  5. Ajoutez dans "Origines JS autorisées" : l'URL de ce fichier (file:// ou votre domaine)
  6. Copiez le Client ID et collez-le ci-dessus
